USAGE INSTRUCTIONS
================================================================================

1. Component Automatically Integrates
   - When ToolCallBox renders a "view_image" tool result
   - Component detects by checking: name === "view_image"
   - Falls back to JSON rendering for other tools

2. Expected Tool Result Format
   {
     "image_data": "iVBORw0KGgo...",  // Base64 encoded
     "file_path": "/path/to/image.png",
     "mime_type": "image/png"
   }

   - Start frontend: npm run dev
   - Create message with view_image tool
   - Verify component displays correctly
   - Test dark mode toggle
   - Test responsive on mobile device

### 修复详情

#### 1. TaskProgressPanel.tsx (Line 120-156)

**问题**: `allOptions` 数组在 `useCallback` 依赖数组中，导致每次渲染都变化

**修复方案**: 将 `allOptions` 数组移到 `handleKeyDown` useCallback 函数内部定义

**代码修改**:
```typescript
// 修复前
const allOptions = [
  { id: null, content: "全部", status: null },
  ...tasks.map((t) => ({ id: t.id, content: t.content, status: t.status })),
];

const handleKeyDown = useCallback(
  (e: React.KeyboardEvent) => {
    switch (e.key) {
      case "ArrowDown":
        setFocusedIndex((prev) => Math.min(prev + 1, allOptions.length - 1));
        break;
      // ...
    }
  },
  [isOpen, focusedIndex, allOptions, onSelect] // ❌ allOptions 每次渲染都变化
);

// 修复后
const handleKeyDown = useCallback(
  (e: React.KeyboardEvent) => {
    // 在函数内部定义 allOptions，避免作为 useCallback 依赖
    const allOptions = [
      { id: null, content: "全部", status: null },
      ...tasks.map((t) => ({ id: t.id, content: t.content, status: t.status })),
    ];

    switch (e.key) {
      case "ArrowDown":
        setFocusedIndex((prev) => Math.min(prev + 1, allOptions.length - 1));
        break;
      // ...
    }
  },
  [isOpen, focusedIndex, onSelect, tasks] // ✅ 使用 tasks 而不是 allOptions
);
```

**验收**: ✅ ESLint 警告消失

#### 2. ChatInterface.tsx (Line 109:6, 166:15, 166:43)

**问题 1**: useEffect 缺少 `MIN_HEIGHT` 依赖

**问题 2**: useMemo 复杂表达式 `(stream as any)?.isLoading` 和 `(stream as any)?.error`

**修复方案**:
1. 将常量定义移到 useEffect 函数内部
2. 移除 useMemo 中不必要的复杂表达式依赖

**代码修改**:
```typescript
// 修复前
const LINE_HEIGHT = 24;
const PADDING_Y = 24;
const MIN_HEIGHT = LINE_HEIGHT + PADDING_Y;

useEffect(() => {
  if (isExpanded && containerRef.current && inputPanelRef.current) {
    const calculatedHeight = containerRef.current.clientHeight - TOOLBAR_HEIGHT - 16;
    setExpandedHeight(Math.max(calculatedHeight, MIN_HEIGHT));
  }
}, [isExpanded]); // ❌ 缺少 MIN_HEIGHT

// 修复后
useEffect(() => {
  // 常量定义 inside useEffect to avoid react-hooks/exhaustive-deps warnings
  const LINE_HEIGHT = 24;
  const PADDING_Y = 24;
  const MIN_HEIGHT = LINE_HEIGHT + PADDING_Y;
  const TOOLBAR_HEIGHT = 44;

  if (isExpanded && containerRef.current && inputPanelRef.current) {
    const calculatedHeight = containerRef.current.clientHeight - TOOLBAR_HEIGHT - 16;
    setExpandedHeight(Math.max(calculatedHeight, MIN_HEIGHT));
  }
}, [isExpanded]); // ✅ 无需 MIN_HEIGHT 依赖
```

```typescript
// 修复前
const subagentList = useMemo(() => {
```

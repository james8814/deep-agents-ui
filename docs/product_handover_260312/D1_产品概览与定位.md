## 1. 产品简介

Deep Agents UI 是一个基于 Next.js 的聊天界面应用，用于与 [deepagents](https://github.com/langchain-ai/deepagents)（LangGraph 增强型 AI 代理）进行交互。它通过 `@langchain/langgraph-sdk` 连接 LangGraph 服务器，实时渲染 Agent 的执行过程，包括工具调用、文件查看、任务规划和人机交互中断审批。当前服务于 PMAgent（AI 产品经理助手）场景，帮助产品经理将模糊的产品想法转化为结构化的可执行文档。

## 3. 用户角色总览

| 角色 | 代码标识 | 说明 | 核心权限 |
|------|---------|------|---------|
| 普通用户 | `User` | 注册登录后使用系统的用户 | 发送消息、上传文件、审批中断、查看任务和文件 |
| AI 助手 | `assistant` | LangGraph Agent | 执行工具、调度子代理、生成文档 |

来源: `src/types/auth.ts`, `src/contexts/AuthContext.tsx`

> 注意：当前代码中未发现多角色权限体系（如管理员），所有注册用户权限相同。

## 5. 产品方向假设 [推断，需验证]

基于功能集合推断，本产品定位为：
- **面向产品经理的 AI 协作工具**：通过 OPDCA 工作流辅助 PM 完成需求分析、PRD 撰写、设计评审等工作
- **Deep Agents 的标准前端**：作为 deepagents 框架的通用聊天界面，可适配不同的 Agent 后端
- **内部工具属性**：从认证方式（用户名密码注册）和部署方式（Docker）推断为团队内部或 B2B 场景

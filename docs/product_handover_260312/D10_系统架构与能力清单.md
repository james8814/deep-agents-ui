### 架构分层

```
┌─────────────────────────────────────────────────────────────┐
│                    用户界面层 (UI Layer)                       │
│  page.tsx → ChatInterface → ChatMessage/ToolCallBox/...     │
│  ThreadList → 会话管理    ContextPanel → 任务/文件           │
├─────────────────────────────────────────────────────────────┤
│                    状态管理层 (State Layer)                    │
│  useChat (核心) → useStream (LangGraph SDK)                 │
│  useThreads (SWR) → Thread 分页                              │
│  AuthContext → 认证状态   ThemeContext → 主题状态            │
│  nuqs → URL 参数状态                                         │
├─────────────────────────────────────────────────────────────┤
│                    服务通信层 (Service Layer)                  │
│  ClientProvider → LangGraph SDK Client (WebSocket)          │
│  fetchInterceptor → Bearer Token 自动注入                   │
│  api/client.ts → Auth Server HTTP 客户端                     │
│  api/upload.ts → 文件上传 (XHR)                              │
├─────────────────────────────────────────────────────────────┤
│                    基础设施层 (Infra Layer)                    │
│  middleware.ts → Next.js 路由保护                            │
│  ClientInitializer → 全局初始化                              │
│  ErrorBoundary → 错误捕获                                    │
│  localStorage → 配置/Token/主题持久化                       │
└─────────────────────────────────────────────────────────────┘
           │                            │
           ▼                            ▼
    Auth Server (:8000)         LangGraph Server (:2024)
    - 用户认证                   - Agent 执行
    - 文件上传                   - 流式通信
    - Token 管理                 - 状态持久化
```

### Provider 层级（依赖关系）

```
ClientInitializer (加载 fetchInterceptor)
  └── AuthProvider (用户认证状态)
      └── AuthGuard (路由保护)
          └── NuqsAdapter (URL 状态管理)
              └── AntdProvider (Ant Design 主题)
                  └── ClientProvider (LangGraph SDK Client)
                      └── ChatProvider (聊天状态)
                          └── 页面组件
```

来源: `src/app/layout.tsx`

## 3. 性能与容量线索

| 配置项 | 值 | 代码位置 | 含义 |
|--------|---|---------|------|
| 文件大小限制 | 100MB | `src/api/upload.ts:MAX_UPLOAD_SIZE` | 单文件上传最大 100MB |
| 线程分页大小 | 由 `limit` 参数控制 | `src/app/hooks/useThreads.ts` | SWR 无限分页 |
| Token 刷新重试 | 最多 3 次 | `src/api/client.ts` | 防止刷新循环 |
| HTTP 重试 | 最多 3 次，指数退避(100/200/400ms) | `src/api/client.ts:fetchWithRetry` | 网络错误容错 |
| Token 过期缓冲 | 30 秒 | `src/contexts/AuthContext.tsx:isTokenExpired` | 提前 30 秒视为过期 |
| 动画帧率目标 | 55+ FPS | `src/app/hooks/useAnimationOrchestra.ts` | 动画性能基线 |
